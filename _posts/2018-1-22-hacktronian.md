# Installation in Linux :

```terminal
$ git clone https://github.com/thehackingsage/hacktronian.git
```
```terminal
$ cd hacktronian
```
```terminal
$ chmod +x install.sh
```
```terminal
$ ./install.sh
```

That's it.. type `hacktronian` in terminal to execute the tool.
